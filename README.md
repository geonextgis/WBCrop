# WBCrop: A manually curated georeferenced dataset of 18 crop types across West Bengal, India

A comprehensive crop classification dataset focusing on 18 major crop types across West Bengal, India.

## Dataset

The dataset includes a manually curated georeferenced dataset of **18 crop types**:

- Aman Rice
- Boro Rice
- Aus Rice
- Jute
- Wheat
- Mustard
- Maize
- Potato
- Vegetables
- Flower
- Tobacco
- Sugarcane
- Banana
- Betel Leaf
- Tea
- Pine Apple
- Groundnut
- Others
